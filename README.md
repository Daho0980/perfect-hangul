# PERFECT HANGUL | 완벽한 한글
이 라이브러리는 Fedora 43 KDE Splash에서 제작되었으며, IBus로 사용 가능한 한글 입력기입니다.
기본적으로 보편적인 한글 입력기의 처리 방식을 따를 겁니다.

## 하지만그럼에도
- 지금은 `Super + Space` 또는 `Shift + Arrow` 등의 특수 입력이 들어올 경우 버퍼에 있는 문자가 그냥 사라지는 문제가 있습니다.
  - 그러므로 가급적이면 `Space`를 누른 후에 사용해야 사라지지 않습니다.
- 지금은 문자가 끊어지지 않고 이어져 있을 때 입력한 문자의 역방향으로 재조합하는 기능은 없습니다.
